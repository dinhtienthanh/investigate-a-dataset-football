  1. Which club should I buy. The club should not be the most popular but somehow on the top, interm on media attraction. So, I set the range based on the matchs the club plays & total goals

Data introduction¶

The data source has been extracted into 7 tables:

Table Name Table Description

1.df_country A table containing the country name of soccer teams

2.df_league A table containing the name of all the soccer leagues

3.df_match A table containing all the match details from 2007 to 2016

4.df_player A table containing all player ID and player name for all the teams

5.df_player_atr A table containing additional information of all the players such as attacking ability, defensive ability, strength, etc.

6.df_team A table containing all the team ID and team name

7.df_team_atr A table containing additional information of all the teams such as their attacking and defensive ability, etc.
